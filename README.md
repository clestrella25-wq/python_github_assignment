# Study Time Tracker

This is a simple Python program that calculates your estimated weekly study hours based on a single day's input. It demonstrates basic input handling, math operations, and error handling.

## How to Run
1. Open your terminal or command prompt.
2. Run the command:
   ```
   python app.py
   ```

## Example Output
```
Welcome to my Python program!
How many hours did you study today? 2
You are on track to study 14.0 hours this week.
```
